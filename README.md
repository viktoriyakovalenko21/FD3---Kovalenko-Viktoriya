# Магазин виниловых пластинок Vinyl Wave

## Инструкция по запуску 

### 1 Необходимо скачать проект и установить зависимости

```bash
npm install
```
### 2 Далее запустить проект, используя команду:

```bash
npm run dev
```
### 3 Для проверки сборки проекта:

```bash
npm run build
```
### 4 Для проверки работы логики корзины (запуск тестов):

```bash
npm test
```
